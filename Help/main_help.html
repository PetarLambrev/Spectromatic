Online documentation

This document is intended as an introductory tutorial to using Spectr-O-Matic. More detailed reference information is available in the online documentation. To access it, at the MATLAB command line type

doc specdata

or

doc specdata2D

and then click on a specific command from the table of contents.

To access the documentation for a specific command directly, for example catindex, type

doc specdata.catindex

or click on the command in the Editor window and press F1.

For information on how to install Spectr-O-Matic, consult ReadMe.txt.

Examples

A selection of examples can be found in the Spectromatic folder. Look for “Examples” and the subfolders therein.

Frequently used commands are collected in a file directly accessible from the MATLAB command window:

open spectromaticlib

The file is organized in sections. Use the “Go to” button in the toolbar (ribbon) to navigate to a particular section.
